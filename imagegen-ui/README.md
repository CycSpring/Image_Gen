# ImageGen Studio (桌面图像生成工作室) 🎨

ImageGen Studio 是一个现代、精致且功能强大的桌面 GUI 客户端，专为 OpenAI DALL-E 及兼容的图像模型设计。支持文本生成图像、图像局部编辑（内置涂鸦画布生成蒙版）、批量并发任务生成等核心功能。

本项目已支持 **独立无缝打包与脱离母系统（Codex/imagegen.ps1）运行**，您可以非常方便地将其分享给朋友使用！

---

## 🛠️ 分享与打包指南

如果您想把这个桌面软件打包成单个 `.exe` 文件分享给朋友，请按照以下步骤操作：

### 1. 编译打包 (生成 .exe)

在当前 `imagegen-ui` 目录下打开终端，运行以下命令：

```bash
npm run dist
```

打包完成后，会在项目根目录下生成 `dist/` 文件夹，其中包含：
- **`ImageGenStudio <版本号> - Portable.exe`**：绿色免安装单文件版（**推荐直接分享这个文件给朋友**）。
- **`ImageGenStudio Setup <版本号>.exe`**：标准的 Windows 安装程序。

---

## 💻 朋友使用时的运行环境要求

当您的朋友拿到您打包的 `.exe` 软件后，需要在他们的电脑上满足以下两个简单条件：

### 1. 安装 Python 3
确保系统安装了 Python 3（建议 3.8+ ），并在安装时勾选了 **"Add Python to PATH"**（将 Python 添加到系统环境变量）。
*可以在命令行运行 `python --version` 来验证是否安装成功。*

### 2. 安装 Python 依赖包
在命令行窗口中执行以下命令安装 OpenAI 与图像处理库：

```bash
pip install openai pillow
```
> **提示**：
> - `openai` 用于请求大模型 API 进行图像生成或编辑。
> - `pillow` 用于处理图片缩放及格式压缩。如果不需要使用图片高级缩放功能，可以只运行 `pip install openai`。

## ⚙️ 软件配置与使用说明

1. **API 密钥与 Base URL 设定**：
   - 打开软件后，在左侧的 **“接口凭证与覆盖” (API Credentials)** 面板中，填入他们的 API Key (例如 `sk-xxxx...`) 以及 Base URL (如果使用的是第三方中转 API)。
   
2. **文本生成图像 (Generate)**：
   - 切换到 **“文本生成”** 选项卡，输入提示词，选择保存路径后点击生成。

3. **局部重绘与编辑 (Edit)**：
   - 切换到 **“局部修改”** 选项卡，上传一张底图。
   - 使用内置的画板画笔在图上直接涂抹出需要修改的区域（紫色蒙版），或直接选择外部的黑白蒙版图片。
   - 输入修改提示词，点击执行修改即可。

4. **批量并发生成 (Batch)**：
   - 支持在界面上直接编写 JSON Lines 格式的任务，或选择本地的 `.jsonl` 文件，一键开启多线程并发生成任务。
